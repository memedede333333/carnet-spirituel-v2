# Configuration Projet Carnet Spirituel

## Sauvegarde Git - IMPORTANT ⚠️

Ce projet utilise **DEUX dépôts Git séparés** :

1. **Code principal** : `/Users/aymeri/projets/carnet-spirituel`
   - Repo : `carnet-spirituel-v2`
   
2. **Documentation privée** : `/Users/aymeri/projets/carnet-spirituel/_private_docs`
   - Repo : `carnet-spirituel-docs`

### Procédure de sauvegarde complète

Quand l'utilisateur demande de "sauvegarder", "commit" ou "push", **TOUJOURS** effectuer ces 3 actions :

1. ✅ **Sauvegarde locale** : Créer une copie timestampée du projet
2. ✅ **Git principal** : `git add -A && git commit && git push` dans `/Users/aymeri/projets/carnet-spirituel`
3. ✅ **Git docs** : `git add -A && git commit && git push` dans `/Users/aymeri/projets/carnet-spirituel/_private_docs`

**Ne jamais oublier aucune de ces 3 étapes.**

## Structure du projet

- `app/` : Code source Next.js
- `_private_docs/` : Documentation sensible (dépôt Git séparé, non publié)
- `.env.local` : Variables d'environnement (jamais commité)

## Variables d'environnement importantes

- `SUPABASE_SERVICE_ROLE_KEY` : Pour opérations admin
- `GMAIL_USER` / `GMAIL_APP_PASSWORD` : Pour envoi d'emails
